# Morning Lineup — Daily Cubs Briefing

Static site generator that produces a daily Cubs briefing as a single HTML page, published to GitHub Pages.

**Live:** https://brawley1422-alt.github.io/morning-lineup/

## Tech stack

- Python 3.12, stdlib only (no pip dependencies)
- Vanilla JS (no frameworks, no build step)
- GitHub Pages (static hosting)
- MLB Stats API v1 & v1.1 (no auth required)
- Fonts: Google Fonts CDN (Playfair Display, Oswald, Lora, IBM Plex Mono)

## Key files

| File | Purpose |
|------|---------|
| `build.py` | Main script — fetches MLB Stats API data, renders full HTML with embedded CSS |
| `deploy.py` | Archives previous day's page, commits new `index.html` to GitHub via API |
| `live.js` | Client-side polling for live Cubs game scores + full slate updates |
| `history.json` | Curated "This Day in Cubs History" entries, keyed by MM-DD |
| `index.html` | Generated output (do not edit directly — regenerated by `build.py`) |
| `template.html` | Old CSS prototype — NOT used in the build pipeline |
| `archive/` | Daily snapshots of previous `index.html` files |

## Build & deploy

```bash
# Generate today's briefing
python3 build.py

# Deploy to GitHub Pages (requires GITHUB_TOKEN env var)
export GITHUB_TOKEN="github_pat_..."
python3 deploy.py
```

## Important patterns

- **All CSS lives in `build.py`** as a string constant (`CSS`), not in a separate file. `template.html` is a dead prototype.
- **Zero external dependencies.** Uses only Python stdlib: `urllib`, `json`, `datetime`, `pathlib`, `html`, `zoneinfo`.
- **Timezone:** Central Time via `ZoneInfo("America/Chicago")` — never hardcode UTC offsets.
- **HTML escaping:** Always use `html.escape()` on API-returned strings before injecting into HTML.
- **Live polling:** `live.js` uses adaptive intervals — 20s when games are live, 5min when idle. Uses innerHTML diff check to prevent flicker.
- **Cubs constants:** `CUBS_ID = 112`, `API = "https://statsapi.mlb.com/api/v1"`

## Sections generated by build.py

1. Yesterday's Cubs game (line score, three stars, key plays)
2. Hot/cold hitters (last 7 games via `lastXGames` roster hydration)
3. Around the League (news wire, scoreboard, standings, leaders)
4. Today's slate (full MLB schedule with probable pitchers)
5. Cubs schedule (next 3 games)
6. Injuries (IL status from 40-man roster)
7. Minor league affiliates (Iowa, Knoxville, South Bend, Myrtle Beach)
8. Cubs team leaders (season stats)
9. This Day in Cubs History

## Design

Dark newspaper aesthetic with Cubs colors. Cream paper (#ece4d0), dark ink (#0d0f14), Cubs blue (#0E3386), red (#CC3433), gold (#c9a24a). Collapsible sections via `<details>` elements.
